# 業務自動化システム導入のご提案

## 1. 要件

* **対象**: Amazon（セラーセントラル）、中国ECサイト、ネクストエンジン間の連携とファイル操作。
* **必須要件** Amazonアカウント保護の為、SP-API（Selling Partner API）が必須になります。
* **認証について** Amazonセラーセントラルは認証に手間がかかり、完全自動化の妨げになります。
* **環境・運用**: Mac/スマホから簡単に操作・確認ができること。指定時刻の自動実行。
* **エラー対応**: データ不備などのエラーを自動検知し、適切に通知・リカバリを行うこと。
* **コスト**: ランニングコスト（維持費）を極限まで抑えること。
